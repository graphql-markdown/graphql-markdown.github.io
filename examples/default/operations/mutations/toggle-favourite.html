ToggleFavourite

Favourite or unfavourite an anime, manga, character, staff member, or studio

ToggleFavourite(
  animeId: Int
  mangaId: Int
  characterId: Int
  staffId: Int
  studioId: Int
): Favourites

Arguments​

ToggleFavourite.animeId ● Int scalar​

The id of the anime to un/favourite

ToggleFavourite.mangaId ● Int scalar​

The id of the manga to un/favourite

ToggleFavourite.characterId ● Int scalar​

The id of the character to un/favourite

ToggleFavourite.staffId ● Int scalar​

The id of the staff to un/favourite

ToggleFavourite.studioId ● Int scalar​

The id of the studio to un/favourite

Type​

Favourites object​

User's favourite anime, manga, characters, staff & studios
