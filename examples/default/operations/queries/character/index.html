Character

Character query

Character(
  id: Int
  isBirthday: Boolean
  search: String
  id_not: Int
  id_in: [Int]
  id_not_in: [Int]
  sort: [CharacterSort]
): Character

Arguments​

Character.id ● Int scalar​

Filter by character id

Character.isBirthday ● Boolean scalar​

Filter by character by if its their birthday today

Character.search ● String scalar​

Filter by search query

Character.id_not ● Int scalar​

Filter by character id

Character.id_in ● [Int] list scalar​

Filter by character id

Character.id_not_in ● [Int] list scalar​

Filter by character id

Character.sort ● [CharacterSort] list enum​

The order the results will be returned in

Type​

Character object​

A character that features in an anime or manga
